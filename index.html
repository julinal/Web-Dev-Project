HTML

Hypertext Markup Language (HTML) makes up the structure of every webpage. HTML uses multiple elements to display content on webpages. Examples of HTML elements are:

  • p
  • h1, h2, h3, h4, h5, h6
  • img
  • ul, ol, li
  • div

CSS

Cascading Style Sheets (CSS) is used to style webpages. With CSS, you can change the color, format, font, etc. of a webpage. CSS uses different properties to style a website. Some examples of different CSS properties are color, background-color, height, width, padding, margin, font-size, border, etc. CSS also allows webpages to have animations.

JS

Javascript makes webpages user interactive. Javascript works with Document Object Model (DOM) to manipulate HTML elements and create new elements. DOM uses selectors to select elements using JS. Hover over the Javascript logo to invert the image.
